# SequenceStrandType
_namespace: [Bio.IO.GenBank](N-Bio.IO.GenBank.html)_

A StrandType specifies whether sequence occurs as a single stranded,
 double stranded or mixed stranded.



### Properties

#### Double
Double-stranded (ds).
#### Mixed
Mixed-stranded (ms).
#### None
None - StrandType is unspecified.
#### Single
Single-stranded (ss).
