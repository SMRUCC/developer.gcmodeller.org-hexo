# FolderIdentifiers
_namespace: [Microsoft.Windows.Shell](N-Microsoft.Windows.Shell.html)_

Contains the GUID identifiers for well-known folders.

### Methods

#### GetAllFolders
```csharp
Microsoft.Windows.Shell.FolderIdentifiers.GetAllFolders
```
Returns a sorted list of name, guid pairs for 
 all known folders.

#### NameForGuid
```csharp
Microsoft.Windows.Shell.FolderIdentifiers.NameForGuid(System.Guid)
```
Returns the friendly name for a specified folder.

|Parameter Name|Remarks|
|--------------|-------|
|folderId|The Guid identifier for a known folder.|

_returns: A @"T:System.String" value._
