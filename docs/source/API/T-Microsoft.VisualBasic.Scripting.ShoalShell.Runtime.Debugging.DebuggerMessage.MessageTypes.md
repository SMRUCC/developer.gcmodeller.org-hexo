### Properties

#### CTRL_DEBUGGER_INIT_INFO
调试客户端向服务器返回初始化信息
#### CTRL_GETS_MEMORY

#### CTRL_KILL_SCRIPT
IDE发送终止脚本调试的信号
#### CTRL_MODIFY_VALUE

#### CTRL_PUSH_SCRIPT
IDE向Shoal调试程序推送脚本
#### OUTPUT_ERROR
Shoal向IDE发送错误消息
#### OUTPUT_MESSAGE
Shoal向IDE发送一般的消息
#### OUTPUT_WARNING
Shoal向IDE发送警告消息
