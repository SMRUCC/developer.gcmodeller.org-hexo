# VSyncMode
_namespace: [OpenTK](N-OpenTK.html)_

Enumerates available VSync modes.



### Properties

#### Adaptive
VSync enabled, unless framerate falls below one half of target framerate.
 If no target framerate is specified, this behaves exactly like @"F:OpenTK.VSyncMode.On".
#### Off
Vsync disabled.
#### On
VSync enabled.
