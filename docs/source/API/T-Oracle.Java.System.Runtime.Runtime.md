# Runtime
_namespace: [Oracle.Java.System.Runtime](N-Oracle.Java.System.Runtime.html)_



### Methods

#### maxMemory
```csharp
Oracle.Java.System.Runtime.Runtime.maxMemory
```
Returns the maximum amount Of memory that the Java virtual machine will attempt To use. If there Is no inherent limit Then the value Long.MAX_VALUE will be returned.
_returns: the maximum amount Of memory that the virtual machine will attempt To use, measured In bytes_
