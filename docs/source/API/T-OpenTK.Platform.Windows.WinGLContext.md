# WinGLContext
_namespace: [OpenTK.Platform.Windows](N-OpenTK.Platform.Windows.html)_

Provides methods to create and control an opengl context on the Windows platform.
 This class supports OpenTK, and is not intended for use by OpenTK programs.

### Methods

#### ToString
```csharp
OpenTK.Platform.Windows.WinGLContext.ToString
```
Returns a System.String describing this OpenGL context.
_returns: A System.String describing this OpenGL context._
